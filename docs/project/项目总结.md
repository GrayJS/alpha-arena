## 🚀 如何启动

### 1. 创建数据库
```sql
CREATE DATABASE alpha_arena;
```

### 2. 配置环境变量
```bash
cd backend
Copy-Item env.example .env
# 编辑 .env 设置数据库密码
```

### 3. 启动项目
```bash
npm run dev
```

### 4. 访问应用
- 前端: http://localhost:5173
- 后端: http://localhost:3000
